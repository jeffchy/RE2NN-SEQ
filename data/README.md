**/data** directory contains datasets used in the paper
- ATIS-BIO
- ATIS-ZH-BIO
- SNIPS-BIO

And in each dataset folder, there are files including
- vocab embedding files [e.g. glove.300.emb]
- rule files [e.g. bio.rules.v1.config]
- decomposed automata files [e.g. /automata/IIID.automata.{name}.pkl]
